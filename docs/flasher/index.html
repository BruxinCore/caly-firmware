Instalador via navegador para dispositivos ESP32
Animações GSAP

Conecte seu dispositivo ESP32 ao computador, clique em Instalar e selecione a porta. O processo roda direto no navegador usando Web Serial/USB.

Depois de instalar, reinicie o dispositivo. Se precisar atualizar novamente, repita o processo aqui.

Se o botão não aparecer, use Chrome/Edge no desktop.
